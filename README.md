## 프로젝트 설명

🔗 배포 주소 : 

저축과 재테크를 시작하고, 저축의 긍정적인 경험을 쌓아갈 수 있도록 도와주는 **웰스테크 플랫폼**인 세이블 앱 내에서 저축 경험과 
목표 달성률을 높일 수 있는 락인 포인트를 개발했습니다. **저축 챌린지**라는 재미 요소를 더해 다양한 사용자들과 경쟁하며 
자연스럽게 저축 습관을 기를 수 있습니다.
